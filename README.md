# 🚀 LokalChat Pro + DeepSeek AI

LokalChat Pro — это кроссплатформенное локальное облако и защищенный чат с интеграцией искусственного интеллекта DeepSeek. Превратите любое устройство в центр обмена файлами и мощный ИИ-помощник.

## 🛡 Возможности
- **Автономность:** Локальное облако работает внутри вашей Wi-Fi сети.
- **DeepSeek AI:** Прямой доступ к чату DeepSeek прямо в приложении.
- **Безопасность:** Вход по логину/паролю, локальное хранение данных.
- **Native Power:** Написано на C++ для максимальной скорости и низкого потребления ОЗУ.

## 📲 Поддержка платформ

| Платформа | Формат файла | Особенности |
| :--- | :--- | :--- |
| **Windows** | `.exe` (Портативный) | Авто-запуск сервера при старте |
| **Linux** | `.deb` / `.AppImage` | Поддержка Debian/Ubuntu/Fedora |
| **Android** | `.apk` | Оптимизировано под Redmi и другие смартфоны |
| **iOS** | `.ipa` | Требуется установка через TestFlight или AltStore |

## 🛠 Установка и запуск

### 1. Десктоп (Windows/Linux)
Просто скачайте соответствующий пакет и запустите:
- Windows: откройте `LokalChat.exe`
- Linux: `sudo dpkg -i lokalchat.deb`

### 2. Смартфоны (Android/iOS)
Установите файл и разрешите доступ к локальной сети в настройках телефона.

## 🤖 Интеграция с ИИ
В приложении появилась вкладка **"DeepSeek AI"**. 
- Нажмите на иконку робота.
- Сразу откроется защищенное окно `chat.deepseek.com`.
- Вы можете копировать данные из своего локального хранилища прямо в чат с ИИ для анализа.

## 📂 Структура (C++/Qt)
- `src/` — исходный код ядра (C++).
- `ui/` — интерфейс на QML.
- `assets/` — иконки и стили.
- `storage/` — папка для ваших локальных файлов.
