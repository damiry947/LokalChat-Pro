# 🚀 LokalChat Pro

**LokalChat Pro** — это кроссплатформенное локальное облако и чат. Превратите любое устройство (ПК, сервер, смартфон или планшет) в защищенную точку обмена файлами.

---

## 🛡 Безопасность
- **Автономность:** Данные никогда не покидают вашу Wi-Fi сеть.
- **Авторизация:** Вход по логину и паролю. Ваши файлы увидите только вы.
- **Приватность:** Никаких сторонних трекеров или облачных сервисов.

---

## 📲 Поддержка устройств


| Устройство | Способ запуска |
| :--- | :--- |
| **Windows** | Запуск `start_windows.bat` (авто-настройка) |
| **Linux / macOS** | Запуск `./start_linux.sh` (авто-настройка) |
| **Android (Redmi)** | Через **Termux** (`python app.py`) |
| **Сервер (Debian/Ubuntu)** | Через **Docker** или `systemd` |
| **Планшеты/iOS** | Работают как клиенты через браузер |

---

## 🛠 Быстрый старт

### 1. На компьютере (Linux/Windows)
Просто запустите соответствующий файл из корня папки:
- `start_windows.bat` — для Windows.
- `start_linux.sh` — для Linux.
*Скрипт сам создаст базу данных и установит нужные компоненты.*

### 2. На Android (Termux)
Если хотите сделать сервером телефон:
```bash
pkg update && pkg install python -y
pip install flask flask_sqlalchemy flask_login
python app.py
```

### 3. Регистрация
1. После запуска перейдите по адресу, который выдаст программа (напр. `http://192.168.1.5:5000`).
2. Нажмите **Регистрация** и создайте свой аккаунт.
3. Теперь вы можете загружать файлы в свое защищенное хранилище.

---

## 📂 Структура проекта
- `app.py` — сервер с поддержкой базы данных и авторизации.
- `database.db` — (создается автоматически) хранит пользователей и пароли.
- `uploads/` — защищенное хранилище файлов.
- `templates/` — профессиональный UI (вход, регистрация, чат).
